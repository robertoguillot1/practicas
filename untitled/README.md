## 🧾 Descripción del Proyecto

Esta aplicación móvil fue desarrollada como proyecto parcial para la materia de **Desarrollo Móvil**, utilizando **Flutter** como framework principal.  
Su propósito es ofrecer una herramienta práctica, moderna y funcional para la **gestión de productos (cervezas)**, **control de inventario**, y **registro de ventas**, con una interfaz clara, intuitiva y adaptable.

---

## ⚙️ Tecnologías Utilizadas

- **Flutter** (Framework principal)
- **Dart** (Lenguaje de programación)
- **intl** (Para manejo de formatos de fecha y moneda)
- **Material Design** (Para la interfaz visual)
- **Stateful Widgets** (Manejo de estados dinámicos)
- **Arquitectura MVC simplificada** (Separación por carpetas: models, pages, utils)

## 🚀 Funcionalidades Principales

### 🏠 **Pantalla Principal (Home)**
- Muestra un resumen general del inventario y las ventas.
- Acceso directo a los módulos principales.
- Diseño simple, moderno y responsivo.

---

### 🍺 **Gestión de Productos**
- Permite **agregar nuevas cervezas** al inventario (nombre, tipo, precio y cantidad).
- Opción de **editar o eliminar** productos existentes.
- Visualización ordenada del catálogo con totales actualizados.
- Usa formatos de moneda y fecha localizados en español (🇨🇴).

---

### 💰 **Registro de Ventas**
- Permite **registrar ventas individuales o múltiples productos.**
- Actualiza automáticamente el stock disponible al confirmar una venta.
- Calcula el **total de la venta** y lo muestra formateado en pesos colombianos.
- Registro de **fecha y hora** de cada transacción.

---

### 📜 **Historial de Ventas**
- Muestra todas las ventas realizadas de forma cronológica.
- Cada registro incluye:
  - Fecha y hora exacta.
  - Productos vendidos.
  - Total de la venta.
- Ideal para control administrativo o revisión de reportes.

---

## 🎨 Diseño e Interfaz
- Basado en **Material Design** para una experiencia visual agradable.  
- Uso de **colores suaves, tarjetas (Cards)** y **botones flotantes (FAB)** para acciones rápidas.
- Diseño adaptable a distintos tamaños de pantalla.

---

## 🧠 Aprendizaje y Objetivo Académico

Este proyecto tiene como objetivo aplicar los conocimientos adquiridos durante el curso de **Desarrollo Móvil**, abarcando:
- Manejo del estado en Flutter.
- Implementación de vistas dinámicas.
- Modelado de datos en clases Dart.
- Separación lógica de responsabilidades (MVC).
- Uso de paquetes externos y buenas prácticas de codificación.
